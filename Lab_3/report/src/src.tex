\section{Архитектура программы}

\subsection{Общая структура}
Программа реализует многопроцессную архитектуру с использованием разделяемой памяти для межпроцессного взаимодействия. Основные компоненты системы:

\begin{itemize}
    \item \textbf{ParentProcess} - управляет созданием дочерних процессов и распределением данных
    \item \textbf{SharedMemory} - инкапсулирует работу с разделяемой памятью и семафорами
    \item \textbf{ChildProcess} - отвечает за запуск дочерних процессов
    \item \textbf{ChildProcessor} - обрабатывает данные в дочерних процессах
\end{itemize}

\subsection{Ключевые алгоритмы}

\subsubsection{Синхронизация с помощью семафоров}
Для координации доступа к разделяемой памяти используются два семафора:
\begin{itemize}
    \item \textbf{sem\_producer} - контролирует запись данных
    \item \textbf{sem\_consumer} - контролирует чтение данных
\end{itemize}
