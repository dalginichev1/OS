\section{Архитектура программы}

\subsection{Общая структура}
Программа реализует архитектуру плагинов с использованием динамической загрузки библиотек. Основные компоненты:

\begin{itemize}
    \item \textbf{DynamicLoader} - загрузчик динамических библиотек
    \item \textbf{AbsDerivative} - абстрактный интерфейс для вычисления производных
    \item \textbf{AbsSort} - абстрактный интерфейс для алгоритмов сортировки
    \item \textbf{Конкретные реализации} - различные варианты алгоритмов в отдельных библиотеках
\end{itemize}

\subsection{Ключевые алгоритмы}

\subsubsection{Динамическая загрузка библиотек}
Для загрузки библиотек используются системные вызовы:
\begin{itemize}
    \item \texttt{dlopen} - открытие библиотеки
    \item \texttt{dlsym} - получение указателей на функции
    \item \texttt{dlclose} - закрытие библиотеки
\end{itemize}
